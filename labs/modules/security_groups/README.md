# Security Groups Module

## Overview

Practice security group management using the Neutron service. You will
create security groups, manage ingress/egress rules, and apply them to
instances.

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | Create Security Group | Create a security group |
| 2 | Manage Rules | Add and remove ingress/egress rules |
| 3 | Apply Security Group | Apply a security group to a running instance |
