# Compute Module

## Overview

Practice compute operations using the Nova service. You will launch
instances, resize them, and manage snapshots.

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | Launch Instance | Launch an instance with a specified flavor and image |
| 2 | Resize Instance | Resize an existing instance to a different flavor |
| 3 | Manage Snapshots | Create and manage instance snapshots |
