# Storage Module

## Overview

Practice block storage operations using the Cinder service. You will
create volumes, attach them to instances, and manage snapshots.

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | Create Volume | Create a block storage volume |
| 2 | Attach Volume | Attach a volume to a running instance |
| 3 | Manage Snapshots | Create and manage volume snapshots |
