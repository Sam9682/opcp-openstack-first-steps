# LACP Configuration Module

## Overview

Practice configuring Link Aggregation Control Protocol (LACP) bonds using the
Neutron service. You will create bonded ports, tune LACP parameters, and attach
bonds to compute instances.

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | Create LACP Bond | Create member ports and a LACP bond (802.3ad) |
| 2 | Configure LACP Parameters | Set LACP rate, hash policy, and activity mode |
| 3 | Attach Bond to Instance | Attach the LACP bond to a compute instance |
