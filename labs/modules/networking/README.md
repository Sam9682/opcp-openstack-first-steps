# Networking Module

## Overview

Practice network configuration using the Neutron service. You will create
virtual networks, subnets, and routers.

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | Create Network | Create a virtual network |
| 2 | Create Subnet | Create a subnet within a network |
| 3 | Configure Router | Configure a router and attach it to a subnet |
