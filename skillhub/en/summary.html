Summary & Next Steps

Difficulty: Beginner · Estimated time: 5 min

Congratulations on completing the OpenStack First Steps training path. Here is a recap of everything you have learned.

What You Learned

  • Core Concepts — Projects, services (Nova, Neutron, Cinder, Keystone, Glance), and endpoints.
  • User Management — Creating users in Keycloak and assigning roles and project attributes.
  • Authentication — Authenticating via Keystone using application credentials and managing tokens.
  • Networking — Creating networks, subnets, and routers with Neutron.
  • Security Groups — Creating groups, managing rules, and applying them to instances.
  • Storage — Creating volumes, attaching to instances, and managing snapshots with Cinder.
  • Compute — Launching instances, resizing, and creating snapshots with Nova.

Next Steps

Now that you have the fundamentals, consider exploring:

  • opcp-openstack-automation — The full automation project with Terraform, Ansible, and advanced deployment patterns.
  • OpenStack CLI — The openstack command-line client for direct API interaction.
  • Horizon Dashboard — The web-based UI for managing OpenStack resources visually.
  • Heat Orchestration — Template-based resource orchestration for complex deployments.
