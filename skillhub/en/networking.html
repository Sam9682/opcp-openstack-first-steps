Networking

Difficulty: Intermediate · Estimated time: 20 min

Networks are the foundation of any cloud deployment. In this lesson you will create virtual networks, configure subnets with CIDR validation, and set up routers — all using the Neutron API through the lab framework.

1. Exercise 1 — Create a Network

Use network_manager to create a virtual network named {student_id}-net-network. 

net = self.managers.get("network_manager")

network = net.create(name=f"{student_id}-net-network")
print(f"Network created: {network}")

2. Exercise 2 — Create a Subnet

Create a subnet within your network. The CIDR is validated before the API call is made. Use 10.0.0.0/24 as the address range. 

from labs.core.validators import validate_cidr

cidr = "10.0.0.0/24"
valid, err = validate_cidr(cidr)
if not valid:
    print(f"Invalid CIDR: {err}")

subnet = net.create_subnet(
    network_name=f"{student_id}-net-network",
    name=f"{student_id}-net-subnet",
    cidr=cidr,
)
print(f"Subnet created: {subnet}")

3. Exercise 3 — Configure a Router

Create a router and attach it to your subnet. This enables traffic routing between subnets and external networks. 

router = net.create_router(name=f"{student_id}-net-router")

net.add_router_interface(
    router_name=f"{student_id}-net-router",
    subnet_name=f"{student_id}-net-subnet",
)
print(f"Router created and attached: {router}")

Summary

  • Use network_manager to create networks, subnets, and routers.
  • CIDR notation is validated before any API call.
  • Routers connect subnets and enable external connectivity.
