Compute

Difficulty: Intermediate · Estimated time: 25 min

The Nova compute service manages virtual machine instances. In this lesson you will launch instances, resize them to different flavors, and create snapshots for backup and recovery.

1. Exercise 1 — Launch an Instance

Use compute_manager to launch an instance with a specified flavor and image. Name it {student_id}-compute-instance. 

compute = self.managers.get("compute_manager")

instance = compute.create(
    name=f"{student_id}-compute-instance",
    flavor="s1-2",
    image="Ubuntu 24.04",
)
print(f"Instance launched: {instance}")

2. Exercise 2 — Resize an Instance

Resize your running instance to a different flavor. This is useful when your workload needs more (or fewer) resources. 

result = compute.resize(
    name=f"{student_id}-compute-instance",
    flavor="s1-4",  # target flavor
)
print(f"Instance resized: {result}")

3. Exercise 3 — Manage Snapshots

Create a snapshot of your instance. Snapshots capture the current state and can be used to restore or clone instances later. 

snapshot = compute.create_snapshot(
    instance_name=f"{student_id}-compute-instance",
    snapshot_name=f"{student_id}-compute-snapshot",
)
print(f"Snapshot created: {snapshot}")

Summary

  • Use compute_manager to launch, resize, and snapshot instances.
  • Flavors define the CPU, RAM, and disk allocated to an instance.
  • Snapshots capture the current state for backup and recovery.
  • Missing flavor or image errors are caught with clear messages.
