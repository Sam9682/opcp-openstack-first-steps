Introduction to OpenStack First Steps

Welcome to SkillHub, the hands-on training companion for the opcp-openstack-first-steps project. This course walks you through the fundamentals of OpenStack — from your first authenticated API call to creating instances, networks, volumes, and security groups.

What You Will Learn

By the end of this training path you will be comfortable with:

  • OpenStack core concepts — projects, users, services, and endpoints
  • User Management — creating users in Keycloak and assigning roles
  • Authentication — authenticating via Keystone using application credentials
  • Networking — creating networks, subnets, and routers with Neutron
  • Security Groups — managing firewall rules with Neutron
  • Storage — creating and attaching volumes with Cinder
  • Compute — launching and managing instances with Nova

Prerequisites

  • An OVHcloud Public Cloud project with API access
  • Basic familiarity with a terminal / command line
  • Python 3.9+ installed
  • Docker installed (for the lab container environment)

Quick Start — Validate Your Setup

Run this small script to confirm your environment is ready:

"""Quick setup validation for opcp-openstack-first-steps."""
import os
import sys

required_vars = [
    "OS_AUTH_URL",
    "OS_APPLICATION_CREDENTIAL_ID",
    "OS_APPLICATION_CREDENTIAL_SECRET",
    "OS_PROJECT_NAME",
]

missing = [v for v in required_vars if not os.environ.get(v)]

if missing:
    print("Missing environment variables:")
    for var in missing:
        print(f"  - {var}")
    sys.exit(1)

print("All required environment variables are set.")
print(f"Auth URL : {os.environ['OS_AUTH_URL']}")
print(f"Project  : {os.environ['OS_PROJECT_NAME']}")
print("You are ready to begin!")
