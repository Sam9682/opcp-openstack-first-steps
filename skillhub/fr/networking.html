Réseau

Difficulté : Intermédiaire · Durée estimée : 20 min

Les réseaux sont la base de tout déploiement cloud. Dans cette leçon, vous allez créer des réseaux virtuels, configurer des sous-réseaux avec validation CIDR et mettre en place des routeurs via l'API Neutron.

1. Exercice 1 — Créer un réseau

Utilisez network_manager pour créer un réseau virtuel nommé {student_id}-net-network. 

net = self.managers.get("network_manager")

network = net.create(name=f"{student_id}-net-network")
print(f"Réseau créé : {network}")

2. Exercice 2 — Créer un sous-réseau

Créez un sous-réseau dans votre réseau. Le CIDR est validé avant l'appel API. Utilisez 10.0.0.0/24 comme plage d'adresses. 

from labs.core.validators import validate_cidr

cidr = "10.0.0.0/24"
valid, err = validate_cidr(cidr)
if not valid:
    print(f"CIDR invalide : {err}")

subnet = net.create_subnet(
    network_name=f"{student_id}-net-network",
    name=f"{student_id}-net-subnet",
    cidr=cidr,
)
print(f"Sous-réseau créé : {subnet}")

3. Exercice 3 — Configurer un routeur

Créez un routeur et attachez-le à votre sous-réseau. Cela permet le routage du trafic entre les sous-réseaux et les réseaux externes. 

router = net.create_router(name=f"{student_id}-net-router")

net.add_router_interface(
    router_name=f"{student_id}-net-router",
    subnet_name=f"{student_id}-net-subnet",
)
print(f"Routeur créé et attaché : {router}")

Résumé

  • Utilisez network_manager pour créer des réseaux, sous-réseaux et routeurs.
  • La notation CIDR est validée avant tout appel API.
  • Les routeurs connectent les sous-réseaux et permettent la connectivité externe.
