Introduction aux premiers pas OpenStack

Bienvenue sur SkillHub, le compagnon de formation pratique du projet opcp-openstack-first-steps. Ce parcours vous guide à travers les fondamentaux d'OpenStack — de votre premier appel API authentifié à la création d'instances, de réseaux, de volumes et de groupes de sécurité.

Ce que vous apprendrez

À la fin de ce parcours de formation, vous maîtriserez :

  • Les concepts fondamentaux d'OpenStack — projets, utilisateurs, services et endpoints
  • La gestion des utilisateurs — créer des utilisateurs dans Keycloak et attribuer les rôles
  • L'authentification — s'authentifier via Keystone avec des application credentials
  • Le réseau — créer des réseaux, sous-réseaux et routeurs avec Neutron
  • Les groupes de sécurité — gérer les règles de pare-feu avec Neutron
  • Le stockage — créer et attacher des volumes avec Cinder
  • Le calcul — lancer et gérer des instances avec Nova

Prérequis

  • Un projet OVHcloud Public Cloud avec accès API
  • Familiarité de base avec un terminal / ligne de commande
  • Python 3.9+ installé
  • Docker installé (pour l'environnement de lab conteneurisé)

Démarrage rapide — Validez votre configuration

Exécutez ce petit script pour confirmer que votre environnement est prêt :

"""Validation rapide pour opcp-openstack-first-steps."""
import os
import sys

required_vars = [
    "OS_AUTH_URL",
    "OS_APPLICATION_CREDENTIAL_ID",
    "OS_APPLICATION_CREDENTIAL_SECRET",
    "OS_PROJECT_NAME",
]

missing = [v for v in required_vars if not os.environ.get(v)]

if missing:
    print("Variables d'environnement manquantes :")
    for var in missing:
        print(f"  - {var}")
    sys.exit(1)

print("Toutes les variables d'environnement sont définies.")
print(f"Auth URL : {os.environ['OS_AUTH_URL']}")
print(f"Projet   : {os.environ['OS_PROJECT_NAME']}")
print("Vous êtes prêt à commencer !")
